F.A.Q

PERGUNTAS FREQUENTES

O Convertex é um aplicativo de busca de serviços que conecta usuários a uma variedade de profissionais, empresas e vendedores em diferentes nichos, facilitando a localização de soluções para diversas necessidades.

Basta baixar o aplicativo, e começar a explorar uma extensa rede de serviços. Digite suas necessidades na barra de busca ou seja direcionado pelas categorias para encontrar a opção perfeita para você.

O Convertex abrange uma ampla gama de serviços, desde profissionais autônomos a grandes empresas, permitindo que os usuários encontrem soluções para necessidades domésticas, profissionais ou de lazer.

O download e uso básico do Convertex são gratuitos para os usuários. Temos apenas 3 tipos de planos que possam suprir com suas necessidades.

Para se cadastrar como prestador de serviços, acesse a seção "Torne-se um Profissional" no aplicativo ou no site. Siga as instruções para criar um perfil e começar a oferecer seus serviços aos usuários do Convertex.

Atualmente, o Convertex está disponível para download nas plataformas iOS (App Store) e Android (Google Play).
